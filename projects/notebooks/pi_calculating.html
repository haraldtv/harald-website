Calculating pi from random floats¶

  • Generate a list of sets of two random float numbers in the range [-1, 1], so they make up a square of sidelength two.
  • Within this square you can find the unit circle with radius 1
\begin{equation} \begin{split} \text{We know the area of a circle is given by: } A = \pi r^2 \end{split} \end{equation}
  • As the unit circle has a radius of one, we know it must have an area of and the square must have an area of 2 * 2 = 4
  • Using the formula below
\begin{equation} \begin{split} \frac{\text{dots within circle}}{\text{total dots}}= \frac{\text{area of circle}}{\text{area of square}} \implies \frac{\text{dots within circle} \cdot \text{area of square}}{\text{total dots}} = \text{area of circle} \end{split} \end{equation}

Power series for calculating pi¶

\begin{equation} \frac{\pi}{4}=\sum_{i=0}^N \frac{ (-1)^i }{2i+1} \end{equation}

def piCircle(n):
    x_s = []
    x_c = []
    y_s = []
    y_c = []
    for i in range(n):
        a = uniform(-1,1)
        b = uniform(-1,1)

        #Checks if the absolute value (length) of the vector is larger than one. If so it's outside the circle
        if (a**2 + b**2 ) > 1:
            x_s.append(a)
            y_s.append(b)
        else:
            x_c.append(a)
            y_c.append(b)
        
    return (len(x_c)) / (len(x_s) + len(x_c)) * 4

piList = []
start = 10
stepsize = 100
N = 5000

x = np.arange(start, N, stepsize)

for i in range(start, N, stepsize):
    piList.append(piCircle(i))

piConst = np.ones(len(x))
piConst = piConst * np.pi

plt.subplot(1,2,1)
plt.plot(x, piList, color="royalblue", label="Calculated value of pi")
plt.plot(x, piConst, color="orange", label="pi (from numpy)")

plt.xlabel("Iterations")
plt.ylabel("Pi")

plt.legend()

plt.subplot(1,2,2)
piDelta = []
for i in piList:
    piDelta.append(abs(i - np.pi))
plt.plot(x, piDelta)

plt.xlabel("Iterations")
plt.ylabel("Abs delta with pi")

plt.tight_layout()
plt.show()

print("Estimate of pi: ", piCircle(N))
